## 使用元组让方法有多个返回值 

一般要使方法返回多个数据，会创建一个类型来返回多个数据。使得不需要再创建一个类型来返回多个数据。可以返回不同的类型.

```csharp
public (int sum, string str) Test(int a, string b)
{
    return (a, b);
}
var val = Test(2, "3");
Console.WriteLine($"{val.sum.GetType()}: {val.sum}");
Console.WriteLine($"{val.str.GetType()}: {val.str}");

输出结果：
System.Int32: 2
System.String: 3
```
